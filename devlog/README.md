# devlog/

Development logs maintained by Claude Code. Each file tracks changes, decisions, and reasoning for a single feature branch.

See [CONVENTIONS.md](CONVENTIONS.md) for full conventions.

## Quick Reference

| Item | Format | Example |
|---|---|---|
| Devlog file | `devlog/NNNNNN-<branch-name>.md` | `devlog/000001-feat-auth-flow.md` |
| Plan file | `devlog/plans/NNNNNN-NN-<description>.md` | `devlog/plans/000001-01-auth-strategy.md` |
| Timestamp | ISO 8601 with UTC offset | `2026-02-14T10:30-08:00` |
